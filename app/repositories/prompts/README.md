# app/repositories/prompts

Arquivos desta pasta:

- (1) Prompt_visao-geral.txt: instrucao para gerar a secao de visao geral do produto.
- (2) Prompt_requisitos-funcionais.txt: instrucao para gerar requisitos funcionais (RF).
- (3) Prompt_telas-escopos.txt: instrucao para gerar telas/escopo de UI-UX.
- (4) Prompt_requisitos-nao-funcionais.txt: instrucao para gerar requisitos nao funcionais (RNF).
- (5) Prompt_regra-negocio.txt: instrucao para gerar regras de negocio (RB).
- (6) Prompt_integracoes-comp-tecnico.txt: instrucao para gerar secoes de integracoes e componentes tecnicos.
- _prompt_normalizacao_word.txt: prompt de normalizacao de formatacao para texto final antes da exportacao para Word.
