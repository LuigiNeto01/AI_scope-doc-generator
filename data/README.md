# data

Arquivos desta pasta (entrada e saida da pipeline):

- Video.mp4: video de entrada usado para extracao de audio.
- ultimo_audio_extraido.txt: referencia para o ultimo arquivo de audio processado.
- escopo.zip: pacote compactado com artefatos de escopo.

- escopo/05_regra-negocio.txt: secao 5 gerada.
- escopo/06_integracoes-comp-tecnico.txt: secao 6 gerada.

- escopo/_teste_documento_escopo.docx: artefato de teste de documento.
