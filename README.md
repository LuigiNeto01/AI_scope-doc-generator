# POC IA Document

Projeto para transformar reunioes em documento de escopo:

1. Extrai audio de um video.
2. Transcreve o audio com IA.
3. Gera secoes de escopo por prompts.
4. Normaliza o texto final.
5. Exporta o escopo para DOCX.

## Arquivos da raiz

- main.py: ponto de entrada da pipeline completa.
- equirements.txt: dependencias Python do projeto.
- .env.example: modelo para variavel OPENAI_API_KEY.
- .gitignore: regras de ignorar arquivos no Git.
- README.md: visao geral do repositorio.
